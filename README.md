# NewsMonkey -realtime news

## Functionality

- #### Realtime News Top Headlines
  - Click on the `TopHeadlines` button to access the top headlines
  - You can search specifically for a keyword in the `search box`
- #### Realtime all Articles Published
  - Click on the `Everyhting` button to access all the content about a specific keyword `Default Keyword is : ALL`
---
## How to use

- To use the app you need to have [_Nodejs_](https://nodejs.org/en/) and [_Git_](https://git-scm.com/downloads) installed on your system.
- After Installing , open the terminal on your local machine and type the command `git clone https://github.com/AnitKrJha/News-App.git`

- Then type `cd News-App`
- Then run the command `npm i` to install required packages
- Then type `npm start` to open the app as an local host.
